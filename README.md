# Sistema de Leilões

## 📌 Sobre o projeto
Este projeto consiste em um sistema informatizado para uma casa de leilões, permitindo o gerenciamento de produtos disponíveis para venda.

## 🗄️ Banco de Dados
O sistema utiliza um banco de dados MySQL com uma tabela chamada `produtos`, contendo informações como nome, valor e status dos itens.

## 🚀 Tecnologias utilizadas
- Java
- MySQL
- Git e GitHub
